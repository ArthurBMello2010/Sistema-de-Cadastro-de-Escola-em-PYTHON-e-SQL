# 📚 Sistema de Cadastro de Alunos

Sistema de cadastro de alunos desenvolvido em Python com integração ao MySQL.

## 🚀 Funcionalidades

- ✅ Cadastro de alunos
- 🔍 Consulta de registros
- ✏️ Atualização de informações
- ❌ Exclusão de alunos
- ✔️ Validação dos dados informados
- 🔗 Integração com banco de dados MySQL utilizando PyODBC

## 🛠️ Tecnologias utilizadas

- Python
- MySQL
- PyODBC

## 📖 Objetivo

Este projeto foi desenvolvido para praticar a integração entre Python e bancos de dados relacionais, aplicando operações CRUD, validação de dados e manipulação de registros SQL.
